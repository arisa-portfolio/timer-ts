# タイマーアプリケーション　要求仕様書

## 1. 概要

本タイマーアプリケーションは、ブラウザ上で動作し、以下の 2つの機能を提供する。

- 設定した時間を計測するタイマー機能
- 設定した時刻に通知するアラーム機能

ユーザーは画面上で **各機能を切り替えて利用できる。**

## 2. 対象環境

- 使用言語: TypeScript
- 使用ツール: Vite, VSCode
- 動作環境: Google Chrome 最新版, Microsoft Edge 最新版

## 3. 機能仕様

### 3.1 タイマー機能

#### 3.1.1 時間設定

- 時間・分・秒単位で設定できる
- 初期値は「00:00:00」とする
- 最大設定時間は「23:59:59」とする

#### 3.1.2 開始

- 開始ボタン押下でカウントダウンを開始する
- 開始後は残り時間を表示し、1秒ごとに更新する
- タイマー動作中は、一時停止およびキャンセル操作ができる
- 開始中は時間設定できない

#### 3.1.3 一時停止

- 計測中に一時停止できる
- 一時停止中は残り時間を保持する
- 一時停止から再開すると、停止した時間から計測を再開する

#### 3.1.4 キャンセル

- キャンセルボタン押下時はタイマーを停止し、設定時間を初期値（00:00:00）に戻す

#### 3.1.5 完了

- 残り時間が 0 になると完了状態になる
- 設定した時間が経過すると音が鳴る
- 完了音は **停止ボタン押下まで継続** する
- 完了後はタイマーを停止し、新たに時間を設定できる

#### 3.1.6 状態保持

- タイマー動作中にページを更新した場合でも、タイマー状態を保持し計測を継続する

### 3.2 アラーム機能

#### 3.2.1 アラーム設定

- **24時間表記** で、時間と分単位で設定できる
- アラーム判定には **日本標準時刻（JST）** を使用する
- 登録可能数は最大 5 件とする
- 同じ時刻のアラームは複数登録できない
- 登録済みアラームが 5 件の場合、新規登録はできない
- 記憶したアラームを有効化・無効化できる

#### 3.2.2 保存

- 登録したアラーム情報はローカルストレージへ保存する
- ブラウザを閉じても設定を保持する

#### 3.2.3 通知

- 有効状態のアラームのみ通知する
- 設定時刻になると音を鳴らす
- 停止ボタン押下で通知音を停止する
- スヌーズボタン押下時は **5分後に再通知** する
- スヌーズは通知中のみ利用できる

#### 3.2.4 状態保持

- ページ更新後も、保存済みアラームおよび有効／無効状態を復元する

## 4. 制約事項

| 項目 | 内容 |
| --- | --- |
| 保存方式 | ローカルストレージを使用 |
| 保存数 | 最大 5 件 |
| 時間精度 | 1 秒単位 |
